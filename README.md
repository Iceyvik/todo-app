# Todo App

A modern and responsive Todo Application built with React and Vite.

## Features

- Add new tasks
- Mark tasks as completed
- Delete tasks
- Task statistics dashboard
- Search and filter tasks
- Priority and category support
- Dark/Light theme
- Responsive design

## Tech Stack

- React
- Vite
- JavaScript
- CSS

## Installation

```bash
npm install
npm run dev
